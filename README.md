# Store Rating Platform

A full-stack web application where users can browse stores, submit ratings, admins manage stores/users, and store owners view analytics.

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS v4
- React Router
- Axios

### Backend
- NestJS
- TypeORM
- PostgreSQL
- JWT Authentication

## Features

### Authentication
- Register / Login
- Role-based access

### Roles
- Normal User
- Admin
- Store Owner

### User
- Browse stores
- Search stores
- Rate stores

### Admin
- Dashboard stats
- Manage users
- Manage stores

### Store Owner
- View average ratings
- View customer ratings
